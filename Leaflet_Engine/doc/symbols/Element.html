Class Element

Prototype for designing Elements. Elements are object that can be added to a layer, and which can be displayed on the map. Elements object are containers for leaflet layers objects like Marker, Line, PolyLine, Polygon, etc... An Element object should only contains leaflet vector layers and leaflet Markers. Other layers are not supported and are not considered as Element. Leaflet Vector Layers objects are defined here Marker leaflet object is defined here
Defined in: leafletEngine.js.
